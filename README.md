# STEEL AEGIS 199X

ブラウザで動作する、1990年代アーケード風の縦スクロールシューティングゲームです。  
GPT-5.4 + Codex で開発しており、作者は1行たりともコードもドキュメント類も書いていません。

## 特徴

- 縦画面レイアウトのスクロールシューティング
- プレイヤーショット、ボム、低速移動、被弾、残機、シールドを実装
- 通常敵、中型敵、ボス、アイテム出現を含む1ステージ構成

## 動作環境

- Node.js 18 以上を推奨
- npm 9 以上を推奨
- 最新の Chrome / Edge / Safari / Firefox

## セットアップ

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで表示されたローカル URL を開くとゲームを開始できます。

## ビルド

本番ビルドを作成します。

```bash
npm run build
```

ビルド結果をローカル確認する場合は次を実行します。

```bash
npm run preview
```

## 遊び方

タイトル画面で `Enter` または `Z` を押すと出撃します。  
ステージを進み、敵を撃破しながらボスの撃破を目指します。

### 操作方法

- 移動: `Arrow Keys` / `WASD`
- ショット: `Z` / `Space`
- ボム: `X`
- 低速移動: `Left Shift`
- ポーズ: `Esc`
- 決定: `Enter` / `Z`

### ゲームルール

- ショットは押しっぱなしで連射できます
- 低速移動中は細かい回避がしやすくなり、自機の判定表示が出ます
- 被弾するとシールドが減少し、シールドが尽きた状態でさらに被弾すると残機を失います
- ボムを使うと敵弾を消しつつ短時間安全を確保できます
- アイテム取得でショット強化、シールド回復、ボム補充が行えます

## 開発用コマンド

- `npm run preview`: ビルド結果の確認
